# Tech Stack

## Frontend – Astro with React for interactive components

- Astro 5 enables building fast, efficient pages and applications with minimal JavaScript.
- React 19 delivers interactivity where it is needed.
- TypeScript 5 provides static type checking and better IDE support.
- Tailwind CSS 4 allows convenient styling of the application.
- shadcn/ui supplies an accessible library of React components that will form the basis of the UI.

## Backend – Supabase as an all-in-one backend solution

- Provides a PostgreSQL database.
- Offers SDKs in multiple languages that act as Backend-as-a-Service.
- Is an open-source solution that can be hosted locally or on your own server.
- Comes with built-in user authentication.

## CI/CD and Hosting

- GitHub Actions for creating CI/CD pipelines.
- Custom VPS hosting of the application via a Docker image.
